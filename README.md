# Генератор красивых EVM-кошельков (Vanity Address Generator)

Генерирует **Ethereum-адреса** с визуально привлекательными паттернами: `0x000`, `0xdead`, `0x6969` и др.

> Работает оффлайн · Безопасно · Быстро

## Особенности
- Поиск по 10+ шаблонам
- Сохранение в `vanity_wallets.txt`
- Поддержка checksum-адресов
- Простая настройка паттернов

## Установка
pip install eth-keys pycryptodome

## Запуск
python vanity_evm_generator.py
